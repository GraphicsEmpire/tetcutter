Aggregator base class. 

An aggregator for collecting operations coming from multiple sources and executing them serially on a single thread. operation\+\_\+type must be derived from \hyperlink{classtbb_1_1interface6_1_1internal_1_1aggregated__operation}{aggregated\+\_\+operation}. The parameter handler\+\_\+type is a functor that will be passed the list of operations and is expected to handle each operation appropriately, setting the status of each operation to non-\/zero.
