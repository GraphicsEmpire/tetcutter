\subsection{Detailed Description}
Implements a sleeping loop to wait for the mutex to unlock.

\begin{DoxyParagraph}{Purpose}
Since this class puts the thread to sleep for short intervals, you can use this class for most of your mutexes. Especially for locking any high level resources where any one operation on the resouce consumes many C\+P\+U cycles. The purpose of this mutex is to reduce the number of C\+P\+U cycles spent in idle loops. All \hyperlink{classLoki_1_1SleepLevelMutex}{Sleep\+Level\+Mutex}\textquotesingle{}s should have higher levels than all your \hyperlink{classLoki_1_1SpinLevelMutex}{Spin\+Level\+Mutex}\textquotesingle{}s.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Dependence on Spin\+Level\+Mutex}
This utilizes \hyperlink{classLoki_1_1SpinLevelMutex}{Spin\+Level\+Mutex} so it does not have to re-\/implement the Do\+Try\+Lock and Do\+Unlock functions the same way. All it really needs is a Do\+Lock function and the amount of time it should sleep if an attempt to lock a function fails. 
\end{DoxyParagraph}
