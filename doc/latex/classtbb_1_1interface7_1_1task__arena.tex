\subsection{Detailed Description}
1-\/to-\/1 proxy representation class of scheduler\textquotesingle{}s arena Constructors set up settings only, real construction is deferred till the first method invocation Destructor only removes one of the references to the inner arena representation. Final destruction happens when all the references (and the work) are gone.
