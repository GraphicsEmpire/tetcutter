\subsection{Detailed Description}
You can place an instance of this as a local variable inside a function to lock a single mutex. It will lock the mutex if no error occurs, or throw if one does happen. When the function ends, the destructor will determine if it needs to unlock the mutex. This R\+A\+I\+I technique insures the mutex gets unlocked even when exceptions occur.
