\subsection{Detailed Description}
Finite state machine. 

/-\/-\/-\/-\/-\/-\/-\/-\/-\/-\/-\/---\textbackslash{} / V 0 --$>$ 1--$>$ ptr --$>$ -\/1 $^\wedge$ $\vert$ $\vert$ V ptr$\vert$1
