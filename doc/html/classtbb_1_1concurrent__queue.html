Detailed Description

template<typename T>
class tbb::concurrent_queue< T >

A high-performance thread-safe queue.

Multiple threads may each push and pop concurrently. Assignment and copy construction are not allowed.

Member Typedef Documentation

template<typename T>
typedef std::ptrdiff_t tbb::concurrent_queue< T >::size_type

Integral type for representing size of the queue.

Note that the size_type is a signed integral type. This is because the size can be negative if there are pending pops without corresponding pushes.
