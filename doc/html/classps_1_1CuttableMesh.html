void ps::CuttableMesh::applyTransformToMeshThenResetTransform ( )

This is a convenient function to aid in transforming original volume meshes and placing them in the right projection view. Later the transform is applied to all mesh nodes and then the transform itself is reset.

int ps::CuttableMesh::convertDisjointPartsToMeshes ( vector< CuttableMesh * > &  vOutNewMeshes)

First finds all disjoint parts of the mesh and then produces new cuttablemesh nodes out of those disjoint pieces. Main mesh will be part of the newly created meshes so the original cuttable mesh node can be discarded if desired.
